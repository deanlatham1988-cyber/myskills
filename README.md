# My Skills

A collection of custom Claude skills and reference materials for automating workflows across MORPh marketing, Obsidian, Outlook, Asana, Gamma, and general writing.

## MORPh Event Marketing

One-command and specialist skills for the MORPh event marketing pipeline — 8-week cadence, 11 emails, 7 social posts, 6 carousel PDFs, NEL packs.

| Skill | What it does |
|---|---|
| [Event Campaign](ai-skills/Event%20Campaign.md) | One-command orchestrator — generates the entire event marketing campaign end-to-end and pushes to SharePoint/SocialBee via Zapier |
| [Event Marketing Generator](ai-skills/Event%20Marketing%20Generator.md) | Coordinator skill that collects the event brief and routes to the three specialist skills below |
| [Event Email Copy](ai-skills/Event%20Email%20Copy.md) | 3 registration emails + 5 delegate engagement emails/SMS, run through brand voice and humaniser |
| [Event Social Copy](ai-skills/Event%20Social%20Copy.md) | 4 company LinkedIn posts + 3 NEL ready-to-post content packs |
| [Event Visual Briefs](ai-skills/Event%20Visual%20Briefs.md) | Design specs for Ariyan (Illustrator/Photoshop) for all 7 event visual assets |
