# My Skills

A collection of custom Claude skills and reference materials for automating workflows across Obsidian, Outlook, Asana, Gamma, and general writing.

## Skills

| Skill | What it does |
|---|---|
| [Email Responder](ai-skills/Email%20Responder.md) | Batch-processes unread Outlook emails, summarises each, and drafts replies in Dean's voice |
| [Daily Note Setup](ai-skills/Daily%20Note%20Setup.md) | Prepends a structured daily template to the current Obsidian daily note |
| [Meeting Transcript Analyzer](ai-skills/Otter%20AI%20Skill.md) | Turns meeting transcripts into structured overviews with follow-up email, Asana task, and brainstorm options |
| [Obsidian Capture](ai-skills/Obsidian%20Capture.md) | Saves information to the Obsidian vault with type-aware formatting and related note linking |
| [Dean Language Style](ai-skills/Dean%20Language%20Style.md) | Enforces Minto Pyramid structure, SCQ openings, banned filler language, and academic-professional register |
| [Toolkit Gamma](ai-skills/Toolkit%20Gamma.md) | Converts .docx training documents into Gamma slide decks with exact wording preservation |
