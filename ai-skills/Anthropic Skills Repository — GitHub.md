*The official Anthropic public GitHub repository containing example Claude skills — the canonical starting point for finding, adapting, and publishing skills.*

## Overview

The Anthropic skills repository at [github.com/anthropics/skills](https://github.com/anthropics/skills) is a public collection of Anthropic-created skills that can be used directly or adapted as templates. It is the reference implementation for how skills should be structured, named, and documented.

The skills/ subfolder (https://github.com/anthropics/skills/tree/main/skills) is the main area of interest — each skill lives in its own kebab-case named folder containing a SKILL.md file and optional scripts/, references/, and assets/ subfolders.

## How to Use This Repository

The recommended workflow is to clone or download the repository, identify a skill that matches your use case, adapt the SKILL.md and any scripts to your own workflow, then upload the folder to Claude.ai via Settings > Capabilities > Skills (or place it in the Claude Code skills directory).

If building from scratch, the repository serves as a pattern library — real examples of how to write effective YAML frontmatter descriptions, how to structure multi-step instructions, and how to organise reference files.
